# 题目描述
请完成一个函数，输入一个二叉树，该函数输出它的镜像。

例如输入：

     4
   /   \
  2     7
 / \   / \
1   3 6   9
镜像输出：

     4
   /   \
  7     2
 / \   / \
9   6 3   1

 
``` cpp
示例 1：

输入：root = [4,2,7,1,3,6,9]
输出：[4,7,2,9,6,3,1]
 
```

限制：

0 <= 节点个数 <= 1000



0 <= 节点个数 <= 10000

# 思路(C++)

## 前序遍历的变形
- 问题分解：
  二叉树的镜像核心问题就是将root左边的所有节点都放在右边，右边的所有节点都放在左边，所以可以通过递归来实现

- 原子问题代码：
``` cpp
交换root的左右子节点

对左右子节点做相同操作
```

- 结束条件：
  - 遍历到root为空时结束循环

### 复杂度
时间复杂度：O(n)， 遍历n个树节点

空间复杂度：O(1)
