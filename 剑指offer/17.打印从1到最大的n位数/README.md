# 题目描述
输入数字 n，按顺序打印出从 1 到最大的 n 位十进制数。比如输入 3，则打印出 1、2、3 一直到最大的 3 位数 999。

``` cpp

示例 1:

输入: n = 1
输出: [1,2,3,4,5,6,7,8,9]
``` 

说明：

用返回一个整数列表来代替打印
n 为正整数


# 思路(C++)

## 因为不用考虑整数
所以挨个打印10^n-1个数字

## dfs递归遍历
字符串打印的话可以对每一位枚举0-9

### 复杂度
时间复杂度：O(10^n)
空间复杂度：O(1)
